Server-Only Mods (NOT included in client pack):
================================================

These mods are installed on the server but should NOT be included in the client mod pack:

- anti-xray-1.3.1-Fabric-1.20.1.jar (Server-side anti-cheat)
- atlas-1.3.0+1.20.jar (Server-only map generation)
- biomespawnpoint-1.20.1-2.2.jar (Server-side spawn management)
- ftbbackups2-fabric-1.20-1.0.23.jar (Server backup tool)
- PlanFabric-5.5-build-2461.jar (Server analytics/monitoring)
- ledger-1.2.8.jar (Server-side logging)
- lithium-fabric-mc1.20.1-0.11.3.jar (Server-side performance)
- krypton-0.2.3.jar (Network optimization, server-side)
- Neruina-2.2.8-fabric+1.20.1.jar (Server-side exception handling)
- global_packs-fabric-1.19.4-1.16.1_fabric.jar (Server-side resource pack management)

Note: When adding mods to Packwiz, make sure to exclude these server-only mods!
