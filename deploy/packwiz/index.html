Join the Fabric 1.20.1 Olympus server with Create, Steam Rails, FTB Teams, and more.

Fast Option: Auto Installer

  1. Download packwiz-installer-bootstrap.exe and place it in your launcher folder (Prism, MultiMC, GDLauncher, etc.).
  2. Open a command prompt inside that folder.
  3. Run:
    ./packwiz-installer-bootstrap.exe https://mc.chroma-scales.com/packwiz/pack.toml
  4. Start the new profile the installer creates and connect to mc.chroma-scales.com.

Direct Download (.mrpack)

Want a file you can double-click in the Modrinth App or Prism?

Download Olympus-1.0.0.mrpack

  1. Save the file somewhere easy to find.
  2. In your launcher choose Import or Add Instance and select the file.
  3. Launch the instance and join mc.chroma-scales.com.

Before You Join

  • Use Java 17 or newer.
  • Allocate at least 6 GB of RAM (8 GB preferred for Distant Horizons).
  • Keep shaders disabled on first launch; add them later if performance is good.
  • Need help? Ask in the Olympus Discord and mention you followed these steps.
