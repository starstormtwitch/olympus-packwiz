Olympus Modpack

Fabric 1.20.1 · Create · Steam Rails · FTB Teams · Join at mc.chroma-scales.com

Download the Pack

Use the same file in every launcher:

Download Olympus-1.0.0.mrpack

Download Olympus-1.0.0-curseforge.zip

Advanced users can sync directly via Packwiz:

java -jar packwiz-installer-bootstrap.jar https://mods.chroma-scales.com/pack.toml

Modrinth App

  1. In Modrinth App, click Add Instance ? Import from file.
  2. Select the downloaded .mrpack.
  3. Click Play and join mc.chroma-scales.com.

Prism Launcher

  1. Create a 1.20.1 instance and launch it once.
  2. Right-click ? Folder ? Open .minecraft.
  3. Drop packwiz-installer-bootstrap.jar in the folder and run:
    java -jar packwiz-installer-bootstrap.jar https://mods.chroma-scales.com/pack.toml
  4. Launch the instance and connect.

MultiMC / PolyMC

  1. Add Instance ? Import from Zip and choose the .mrpack file.
  2. Launch; the mods are installed automatically.
  3. Join mc.chroma-scales.com.

CurseForge App

  1. Click Create Custom Profile ? Import.
  2. Select the downloaded .mrpack.
  3. Start the new profile and connect.

Packwiz CLI

  1. Place packwiz-installer-bootstrap.jar in your instance folder.
  2. Run:
    java -jar packwiz-installer-bootstrap.jar https://mods.chroma-scales.com/pack.toml
  3. Launch the generated profile.

Before You Join

  • Java 17 or newer.
  • Allocate 6–8 GB RAM.
  • Disable shaders on first launch.
  • Need help? Ping the Olympus Discord mods.
