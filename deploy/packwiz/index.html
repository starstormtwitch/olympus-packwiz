Quick Install via Packwiz

Open your Minecraft instance folder (Prism, MultiMC, etc.) and run:

./packwiz-installer-bootstrap.exe https://mc.chroma-scales.com/packwiz/pack.toml

This keeps your client in sync with the server automatically.

Modrinth / GUI Import

Prefer a GUI? Import the latest Olympus-1.0.0.mrpack into the Modrinth app or Prism launcher.

  • Launcher ? Import Instance
  • Point to the .mrpack from the repo’s dist/ folder
  • Launch, then join mc.chroma-scales.com
