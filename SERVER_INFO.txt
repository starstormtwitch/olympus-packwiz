7. Enter server name: Olympus

9. Click "Done" and join the server!
10. Prefer CLI? Run packwiz-installer-bootstrap.exe https://mc.chroma-scales.com/packwiz/pack.toml from your instance folder to auto-sync mods.

Method 2: Manual Installation
------------------------------
1. Download and install Fabric Loader 1.20.1
2. Download all mods from the mod pack
3. Place mods in your .minecraft/mods folder
4. Launch Minecraft with Fabric
5. Add the server using the IP above
6. Join the server!

========================================
Troubleshooting:
========================================

If you can't connect:
- Make sure you have the correct mod pack installed
- Check that your Minecraft version is 1.20.1
- Verify that Fabric Loader is installed correctly
- Ensure all mods are loaded (check the mods menu)
- Check your firewall settings
- Verify the server IP and port are correct

If mods are missing:
- Reinstall the mod pack using Modrinth App
- Make sure you downloaded the latest version
- Check that all mods are enabled in the mods menu
